# Internship Frontend Projects

This repository contains three frontend projects built for an internship:

## Projects
1. **Music Player**
   - A simple web-based music player with play/pause, next/previous features.

2. **Calculator**
   - A basic calculator built using JavaScript for arithmetic operations.

3. **Image Gallery**
   - A gallery to view and browse images with a clean UI.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open the project folder in VS Code
3. Open `index.html` in a browser
